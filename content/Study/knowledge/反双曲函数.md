# 反双曲函数

反双曲函数在考研数学一中的出现频率不算高，但一旦出现就是**关键识别点**——认不出来就没法展开，没法展开整道题就卡住。

---

## 定义与对数形式

### 反双曲正弦（最重要）

$$
\operatorname{arsinh}x = \ln(x + \sqrt{1 + x^2}), \quad x \in \mathbb{R}
$$

- **奇函数**：$\operatorname{arsinh}(-x) = -\operatorname{arsinh}x$
- 泰勒展开只有奇次项：$\operatorname{arsinh}x = x - \dfrac{x^3}{6} + \dfrac{3x^5}{40} + O(x^7)$
- 导数：$\dfrac{d}{dx}\operatorname{arsinh}x = \dfrac{1}{\sqrt{1+x^2}}$

> [[反双曲正弦与对数展开极限_2026-06-27|③ arsinh 极限]] 中，识别出 $\ln(x+\sqrt{1+x^2})$ 就是 $\operatorname{arsinh}x$ 是解题第一步。

### 反双曲余弦

$$
\operatorname{arcosh}x = \ln(x + \sqrt{x^2 - 1}), \quad x \geq 1
$$

- 非奇非偶，定义域 $[1, \infty)$
- 考研中出现频率较低

### 反双曲正切

$$
\operatorname{artanh}x = \frac12 \ln\frac{1+x}{1-x}, \quad |x| < 1
$$

- **奇函数**
- 展开：$\operatorname{artanh}x = x + \dfrac{x^3}{3} + \dfrac{x^5}{5} + O(x^7)$
- 注意展开形式很接近 $\arctan x$，系数相同但符号不同

---

## 为什么考研会考反双曲函数

因为 $\sin x, \cos x, \ln(1+x), e^x$ 这些展开太常用了，换一个**看着像但展开不同**的函数来检验你是否真的理解了泰勒展开的原理，而不是只会背公式。

$\operatorname{arsinh}x$ 的陷阱在于：
- 它看起来像 $\ln$ 的复合，但展开用 $\ln(1+u)$ 去推会因 $u$ 的表达式复杂而容易算错
- 它的 $x$ 项系数和 $\ln(1+x)$ 一样（都是 1），导致相减时低阶项消去，需要展开到更高阶

---

## 常用展开对比（$x \to 0$）

| 函数 | 展开到 $x^3$ 项 |
|:---|:---|
| $\ln(1+x)$ | $x - \dfrac{x^2}{2} + \dfrac{x^3}{3} + O(x^4)$ |
| $\operatorname{arsinh}x$ | $x - \dfrac{x^3}{6} + O(x^5)$ |
| $\arcsin x$ | $x + \dfrac{x^3}{6} + O(x^5)$ |
| $\arctan x$ | $x - \dfrac{x^3}{3} + O(x^5)$ |
| $\sin x$ | $x - \dfrac{x^3}{6} + O(x^5)$ |
| $\tan x$ | $x + \dfrac{x^3}{3} + O(x^5)$ |

注意到 $\operatorname{arsinh}x$ 和 $\sin x$ 的 $x^3$ 项系数相同（都是 $-1/6$），因为它们的导数 $\dfrac{1}{\sqrt{1+x^2}}$ 和 $\cos x$ 在 0 处展开的前两项一致。
