# 常用放缩不等式

放缩的核心是找到一个**简单且有界（或可比较）** 的函数来夹住目标函数。

---

## $|\sin t| \leq |t|$ — 局部线性放缩

这是**最有威力的不等式之一**，因为它把震荡函数 $\sin$ 变成了线性函数。

**几何直觉**：单位圆上，弧长 $|t|$ 总是 ≥ 对边长度 $|\sin t|$。

**典型应用**：
- $|x \sin(1/x)| \leq x \cdot |1/x| = 1$ — [[函数有界性判定_2026-06-10|① 有界性判定]]
- $\displaystyle \lim_{x \to 0} \frac{\sin x}{x} = 1$ 的证明就是用这个不等式夹逼

**注意**：它只在 $t$ 接近 0 时比 $|\sin t| \leq 1$ 更紧。当 $|t| > 1$ 时退化为平凡的 $|\sin t| \leq 1$。

---

## $|\sin t| \leq 1$ 与 $|\cos t| \leq 1$

最朴素的放缩，在 $|t|$ 很大时也是唯一的放缩工具（因为 $|\sin t| \leq |t|$ 此时已经松到没有约束力）。

**应用场景**：
- $\left|\dfrac{\sin x}{x}\right| \leq \dfrac{1}{x}$（$x \geq 1$ 时）
- $|x \sin x| \leq x$（但这个界太松，不能证明有界——这正是 [[函数有界性判定_2026-06-10|① 中 $x\sin x$ 无界的原因]]）

---

## 其他常用不等式

| 不等式 | 适用场景 |
|:---|:---|
| $\lvert\sin t\rvert \leq \lvert t\rvert$ | 局部放缩，处理 $\sin(1/x)$ 类 |
| $\lvert\sin t\rvert \leq 1,\ \lvert\cos t\rvert \leq 1$ | 全局放缩 |
| $\lvert a \pm b\rvert \leq \lvert a\rvert + \lvert b\rvert$ | 三角不等式，拆分复杂表达式 |
| $\bigl\lvert \lvert a\rvert - \lvert b\rvert \bigr\rvert \leq \lvert a - b\rvert$ | 反向三角不等式 |
| $\dfrac{2}{\pi}x \leq \sin x \leq x\ (0 \leq x \leq \dfrac{\pi}{2})$ | 特定区间的紧放缩 |
| $e^x \geq 1 + x$ | 指数函数放缩 |
| $\dfrac{x}{1+x} \leq \ln(1+x) \leq x\ (x > -1)$ | 对数函数放缩 |

---

## 放缩的方向陷阱

放缩最怕**方向搞反**：

- **证有界**：需要放大到有界函数，即 $|f(x)| \leq M$
  - 例：$|x\sin(1/x)| \leq x \cdot |1/x| = 1$ ✅（放大了但不超过 1）
  - 反例：$|x\sin(1/x)| \leq x$ ❌（$x$ 在 $(0,\infty)$ 上无界，放了等于没放）

- **证无界**：需要找一个下界趋于无穷，通常不用放缩而是用[[函数有界性判定方法\|数列法]]
