# 常用泰勒展开（考研版）

泰勒展开是考研数学一极限题的**核心通用武器**。等价无穷小是它的特例，洛必达能做的一般都能用泰勒做。

---

## 考研必背展开式（$x \to 0$）

| 函数 | 展开式（到 $x^4$ 阶） |
|:---|:---|
| $\sin x$ | $x - \dfrac{x^3}{6} + \dfrac{x^5}{120} + O(x^7)$ |
| $\cos x$ | $1 - \dfrac{x^2}{2} + \dfrac{x^4}{24} + O(x^6)$ |
| $\tan x$ | $x + \dfrac{x^3}{3} + \dfrac{2x^5}{15} + O(x^7)$ |
| $\arcsin x$ | $x + \dfrac{x^3}{6} + \dfrac{3x^5}{40} + O(x^7)$ |
| $\arctan x$ | $x - \dfrac{x^3}{3} + \dfrac{x^5}{5} + O(x^7)$ |
| $e^x$ | $1 + x + \dfrac{x^2}{2} + \dfrac{x^3}{6} + \dfrac{x^4}{24} + O(x^5)$ |
| $\ln(1+x)$ | $x - \dfrac{x^2}{2} + \dfrac{x^3}{3} - \dfrac{x^4}{4} + \dfrac{x^5}{5} + O(x^6)$ |
| $(1+x)^\alpha$ | $1 + \alpha x + \dfrac{\alpha(\alpha-1)}{2}x^2 + \dfrac{\alpha(\alpha-1)(\alpha-2)}{6}x^3 + O(x^4)$ |
| $\operatorname{arsinh}x$ | $x - \dfrac{x^3}{6} + \dfrac{3x^5}{40} + O(x^7)$ |
| $\operatorname{artanh}x$ | $x + \dfrac{x^3}{3} + \dfrac{x^5}{5} + O(x^7)$ |

> $\operatorname{arsinh}x = \ln(x + \sqrt{1+x^2})$ 的展开需要记住，考场上现场推导浪费时间。

## 阶数判断法则

### 核心原则：展开到**比消不掉的最低次高一阶**

**经典案例**：[[反双曲正弦与对数展开极限_2026-06-27|③ arsinh 极限]]

$$
\lim_{x \to 0} \frac{\ln(1+x) - \operatorname{arsinh}x}{\operatorname{arsinh}x \cdot \ln(1+x)}
$$

- $\ln(1+x) = x - \dfrac{x^2}{2} + \dfrac{x^3}{3} - \cdots$
- $\operatorname{arsinh}x = x - \dfrac{x^3}{6} + \cdots$

两者 $x$ 项相同（都是 $x$），相减后 $x$ 项消去。**只展开到 $x$ 项会得到 $0/0$ 且无法判断**，必须展开到 $x^3$ 项。

### 分式型极限的阶数指南

$$
\lim_{x \to 0} \frac{P(x)}{Q(x)}
$$

记 $P(x)$ 和 $Q(x)$ 的最低次项次数分别为 $p$ 和 $q$。
- 若 $p = q$：极限为 $\dfrac{a_p}{b_q}$（系数比）
- 若 $p > q$：极限为 $0$
- 若 $p < q$：极限为 $\infty$

**展开策略**：分子分母都展开到 $\max(p,q)$ 阶，保证能读取出最低次项的系数。

### $1^\infty$ 型的阶数判断

[[极限1∞型与导数几何意义综合_2026-06-10|② 综合极限]] 的核心技巧：

底数 $f(x) \to 1$，指数 $1/x^k$，则底数要展开到 $x^k$ 项。

例如指数为 $1/x^2$：底数展开到 $\cos x = 1 - \dfrac{x^2}{2} + o(x^2)$ 就够了，$\sin$ 的四次项在此场景中可忽略。

### 变上限积分的展开阶数

若 $f(0) = 0$，$f'(0)$ 存在：

$$
\int_0^x f(t)\,dt = \frac{f'(0)}{2}x^2 + o(x^2)
$$

积分比被积函数**高一阶**。

## 乘积展开的交叉项

两个无穷小相乘时，交叉项不能漏：

$$
(x + a_2x^2 + a_3x^3)(x + b_2x^2 + b_3x^3) = x^2 + (a_2 + b_2)x^3 + (a_3 + b_3 + a_2b_2)x^4 + \cdots
$$

> ③ 题中 $\operatorname{arsinh}x \cdot \ln(1+x)$ 就是这样展开的。
